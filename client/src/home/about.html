  • About the project development:

    What we implemented:

    • The user is, upon arrival, able to specify his/her name/nick. If the nickname is free, i.e. no other user is currently active with the same name, (s)he proceeds, otherwise a new nick must be provided.
    • After the user has identified, (s)he can see a list of chat rooms that're already active.
    • The user is able to join a chat room, and leave a room as well. It is of course also possible to create a new room.
    • Inside a given room, the user is able to send messages to the room, see previous messages, and see new messages appear in real time (without having to refresh the page manually).

    In addition:

    • Each component (controller, factory, etc.) is in a single file. And they're minified and concatenated when prepared for production by using gulp tasks.
    • All external dependencies (AngularJS etc.) are installed by using npm and bower.
    • The code goes through JSHint without warnings. A gulp file is included to ensure that running jshint is easy. A .jshintrc file is also included with the options provided on Myschool.

    And more:

    • The current username and logged in status (along with private messages) is stored in a variable called "theUser."
    • "Minimum" security is provided by checking "theUser" logged in status in each controller and changing the url to the login screen if one's not logged in.
    • Usernames and chatroom names can only be one word and no symbols. We used ng-pattern to implement that.
    • Buttons for logging in, joining or creating a chatroom and sending messages are disabled if empty or invalid in any way.
    • Button titles only appear when they're disabled to provide a possible reason for them being disabled.
    • Leaving a chatroom works by either clicking the "Leave Room" button, by changing the url or by clicking the links on the left. When the url changes, the user is logged out of that room.
    • On the list of useres in a room, the ops are in bold.
    • Private chat didn't work in the end up but one can click the links for the users in a chatroom to open up a private chat. Clicking your own name there will transfer you to the list of all logged on users.
